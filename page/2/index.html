万亿倍成长开篇语

发表于 2017-08-28 | 分类于 学习成长 |
| 字数统计

先上结论，晚点扩充详细内容。
1、无论基础如何，按照改进1%的成长速度，要做出10倍改进至多需要232次。
2、无论基础如何，按照改进10%的成长速度，要做出10倍改进至多需要25次。
3、无论基础如何，按照改进1%的成长速度，要做出一万亿倍改进至多需要2777次改进，每天一次至多7年。
4、无论基础如何，按照改进10%的成长速度，要做出一万亿倍改进至多需要290次改进。
5、无论基础如何，成长速度不变，从0到1跟从1到10或者10到100一样难，需要一样的改进次数，并没有更难或者更容易。
6、根据以上5条结论可知，要想快速到达目标，只有在成长速度上想办法才是正确的途径。
7、以上结论从侧面证明：成功没有捷径，唯有踏踏实实的一个脚步一个脚印的改进；成功也没有看起来的那么艰难，进步一万亿倍也不过需要2777次改进而已。
