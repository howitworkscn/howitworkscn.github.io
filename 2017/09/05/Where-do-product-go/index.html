产品往何处去？

发表于 2017-09-05 | 分类于 产品思考 |
| 字数统计

在读大学的时候曾读过波兰作家显克微支的《你往何处去？》，小说讲的是最早期的基督徒与异教徒的斗争。其中有个场景让我记忆深刻：罗马皇帝为了写诗命人放火烧了罗马城，事后嫁祸给基督徒，派兵全城抓捕基督徒。然后把抓捕到的黎吉亚公主绑到斗牛角上，让勇士威尔史与牛搏斗。
