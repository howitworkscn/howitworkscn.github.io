没有激情的进步都是假成长

发表于 2018-05-16 | 分类于 学习成长 |
| 字数统计

过去一个月，几乎所有的心思都放在了学车，练车上，为的就是今天能够顺利通过科目二。

上周日下午，带着儿子去看考场，发现考场跟平时训练的不一样，但是心情并没有太大的波动。

周二跑去练车，不太顺利，还是一样的犯着一些常见的错误，心里有点忐忑。教练安慰了下，你技术是没问题了，就是心态不稳定。

晚上回家，意识几乎沉睡状态，只是机械的给宝贝洗完澡，开手机给宝贝听，陪他一起睡觉。

睡觉前发生了点小插曲，宝贝不肯刷牙，老婆发脾气，全都怪到我身上来，照常的吵了几句，意识还是沉睡状态，心情并未出现大波动。

开着凯叔的《声律启蒙》陪宝贝睡觉，睡到半梦半醒之间，突然梦到倒车的时候车速太快，然后吓醒了，然后开始睡不着。

也许房子外面的机器声音太吵，也许刚刚小睡了一小会没那么困了，也许太热了，总之就是睡不着。

左翻右侧总害怕又梦到刚刚的场景，最后如此这般折腾了半天最终还是睡着了。

失眠有过，但是因为考试而失眠却是第一次。

虽然跟真正的学霸比起来，自己就是如假包换的学渣，但是放到自己的成长环境中去，自己也算得上别人眼中的学霸。

读书工作的过程中，经历过不少的考试，真正因为担心考试而失眠的场景，应该没有，至少记忆中不存在。

这样说，并不是为了显示自己有多学霸，而是想说，过去的考试，在内心深处——几乎是无一例外当成检验自己学习成果的一次练习，而自己对自己的掌握程度一直都算是心里有数的。

但是这次不同，也从来没有哪一次考试像这次这样如此的不同。这一次，寄托了太多的期待，承载着太多的承诺。

所以，也是第一次让我如此在意一场考试。

早上一切顺利的上了考试车，排队验证的时候还担心自己是第一个考，名单出来后是第二个考放心不少。

前面的那位倒车入库挂了一次，第二次成功通过。

轮到自己，紧张到了极点，在倒车的时候，左腿都是在颤抖的，万幸的是第一把就顺利的通过。

接下来四项连考，有两位在我前面考，一个过了，一个挂了。

轮到我的时候，第一把，坡道定点停车，边线超30里面，直角转弯时候后压线，挂了。

第二把，一切顺利，到直角转弯的时候，感觉要压线，赶紧回方向盘，幸好没有压线，正满怀希望的能够满分通过的时候，语音报直角转弯后轮压线，实际肉眼看到是还没碰到线，事后跟教练说，是有5公分的误差的。

突然感觉好难过，好不甘心！

怎么就压线了呢？怎么就压线了呢？怎么就压线了呢？

漫谈产品

发表于 2017-09-14 | 分类于 产品思考 |
| 字数统计

都知道一个小孩从0到1，从无到有的过程中，萌芽是那个瞬间。

一个产品从0到1，从无到有的过程中，萌芽是什么呢？

那么产品是不是也有那么一个瞬间呢？应该有吧，可能称为灵感，点子，想法等等不一而足。

而这个想法或者灵感的子宫是什么呢？它需要一个温暖的、舒适的环境，充足的营养供给。只有这样才能正常健康的成长成胚胎，直至成为婴儿来到这个世界。

在老婆怀孕的那段时间里，一直在想，一个产品的诞生跟一个小孩的诞生是何其相似啊。作为父母热爱自己的小孩自然是理所应当的，但是反观现在的社会大众，热爱自己的产品的人并不多。

究其原因，可能是太多人是抱着做一个赚钱的产品，赚一笔就走的心态在做产品。而且产品有一点是无法跟小孩相类比的——归属。

孩子的出生天然就是其父母的，而且是一辈子的无法改变的事实。但是产品不一定，产品最开始可能是你的，但是当产品遭遇了各种好的坏的事情后，可能就不再是你的了。

尽管曾经有着你的基因在里面，但是终有一天会离开你，就像小孩终有一天离开你去闯荡自己的天地一样，与小孩不同的是，产品一旦离开你就不再是你的产品了。

不要停下来

发表于 2017-09-13 | 分类于 学习成长 |
| 字数统计

开始的1000天跑步训练计划，到今天已经进行了60多天，但是上周末，公司组织去增城的白水寨爬山，回来太过劳累，于是没有去跑步，预览之中的事，后面就联系着三天没有跑。而且开了这个头后面就只要自己觉得有点累或者不想跑啊，什么的就真的不再跑了。所以坚持做一件事的话，一定不要在中途停下来，不应该开这种先例。否则后面任何一个理由都能说服你妥协的。

这其实跟人性的运行机制是类似的。有第一次就不会有最后一次了。

比如职业上，刚出校门的应届生，一般会想着，先以低工资拿经验，等有经验后再要求涨工资。如无意外，这样子干上个3年，5年，10年，你的工资水平都只能在中下等水平，基本不会比平均水平好多少。即使你真的非常勤奋，那么也最多比平均水平稍好。

比如家庭暴力，年轻的夫妇，可能会原谅第一次的暴力对待，觉得对方只是太生气了，没控制住自己。而且对方又是那么诚心的道歉，于是就原谅了。这个婚姻要么最终以离婚散场，要么以其中一方的毁灭结束。

我们不一样

发表于 2017-09-11 | 分类于 学习成长 |
| 字数统计

还年轻的那会，总感觉自己平平无奇，每个人都跟自己很像，没什么太本质的区别。

但是这真是个大错误，没有什么比这种看法更糟糕了，几乎毁了生活中的所有乐趣，也毁了自己的自信。

现在回过头来看，我们哪里一样了呢？

五官就不可能是一样的，即使是双胞胎都不会完全一样的。

经历不可能是一样的。

心里想的更加不可能一样的。

家庭生活不可能一样的。

兴趣爱好可以相同，但不可能都相同。

我们就是不一样的，即使我们最初开始生命的瞬间都不是一样的，我们包含了不同的基因。

我们就是不一样的，成长环境，成长经历，所有的东西不可能一样。

我们就是不一样的，即使我们的爱好都一样，那么我们的程度也是不一样的。

我们就是不一样的，即使我们的能力是类似的，但是我们水平是不可能相同的。

所有的所有，每一个细节，每一个瞬间，都是不一样的，这样最终造就了我们最终的不一样，我们最终命运的不一样。
